## 4. 风险与战略备忘（承接 DCP-2）

- **一致性校验不得退化为只过可达性** —— ✅ 守住（validator 双层 + 负例召回 100%）。
- **中转站约束**（Opus-only / 无 Sonnet / 计价虚高 / 长响应）—— 成本含校验偏高、validator 降本待直连 key；已改流式降网关超时风险。
- **成本口径新发现**：校验（opus-4-7 逐条一致性）是成本大头，含校验一轮 ≈ ¥14–26，超 analyze-only 阈 → ② 定稿须先厘清"含/不含校验"口径。
- **跨源仍偏低**：双主题 1/45 真跨源（追求层 ≥30%、非门槛）；选片/提示可迭代。
