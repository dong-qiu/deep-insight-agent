# 成本 / 性能标定（M3-3）

> 2026-05-27 · 基于本会话多次真实运行的数据点综合（非单次受控测量——中转站近期反复超时，
> 且真实扣费以中转站账单为准；本报告为 **Anthropic 列表价估算 ×7.2**，用于**相对量级 + 阈值对照**）。

## 口径与 caveat
- 成本 = `analyze`（opus-4-6）+ `validate`（opus-4-7）；`report-gen` 确定性 **$0**。单份报告 ≈ 一个 topic 一轮。
- **validator thinking 默认 off**（A1 证带思考仅边际提精度、负例召回 98.3%；on 成本数倍且中转站缓存计价虚高）。
- **成本由内容形态主导**，不是报告类型：arXiv 摘要 ~600 字 vs 富正文（Latent Space/Krebs 可达 5 万字）差一个量级。
- 中转站自缓存上报使带思考估算虚高（A1 全量曾估 $21、实则 token 仅值 ~$1）；下表取 thinking-off 干净数。

## 按报告类型的成本区间（thinking-off）
- **brief（一个主题一轮）**：
  - arXiv-clean（短摘要）：analyze ¥0.5–1 + validate ¥1–2 → **~¥2–4**（阈内）。
  - **富多源（长正文）**：analyze ¥4–6 + validate ¥4–7 → **~¥7–13 ⚠️ 超 ¥5 重标阈**。
- **deep_dive**：更多条目 → 按比例更高（富多源易破 ¥15）。
- **initial_digest**：冷启动回填全量，量级最大（未实测）。

## 对照重标阈值（brief¥5 / deep_dive¥15 / initial_digest¥30）
- **arXiv-clean 内容在阈内**；**富多源内容 brief 即超 ¥5（thinking-off 也超）**。
- **根因**：① 富正文（50k 字）灌 analyze 输入大；② F4 分批使大 SYSTEM 提示**重复 N 次**；③ 引用越多 validate 调用越多。
- **结论**：阈值需**内容感知**，或靠下列杠杆把富正文成本压回 —— 不重开阈值，列为 M3-3 输出的待调项。

## 成本杠杆（按性价比）
1. **analyze body 上限**（最高杠杆）：`MAX_BODY_CHARS=50k` 对分析过于慷慨；富正文截到 ~8–12k 字喂 analyze（保留首部+关键段）可大幅降本，且不太损洞察（abstract/导语信息密度最高）。env 化便于调。
2. **thinking-off 默认**（已采）：validate 成本数倍差。
3. **分批 SYSTEM 开销**：prompt caching 可省重复 system，但中转站不命中 → 待直连 key。
4. **直连 key + Sonnet 分析**（将来）：analyze 输入价 1/1.7 + 缓存命中 → 整体大幅降。

## 性能（P50 时延 · charter ≤10 分钟）
- live-f4：12 条富正文 / 9 批 analyze = **290s（4.8 min）**；多源全跑（analyze+validate）≈ 5–10 min。
- **当前规模在 ~10 min 目标内，但富正文 + 分批已接近上限**；条目/正文再大需分批并发或 body 上限（杠杆 1 同时降时延）。

## 落实验证（同日 · analyze body 上限）
实现 `ANALYZE_BODY_CHARS=10k`（renderItems 截短富正文喂分析）后，同配置多源复测：
- **成本 $1.86 → $0.99（~砍半）**：analyze 输入 73991→31850 tok（↓57%）、analyze $0.51→$0.30、validate $1.35→$0.69。
- **可达性 78.9%**（run3 88.6%）：在 run-to-run 噪声内；**截断为全文前缀、quote 取自所见 ⊂ 全文 → 可达性 safe by construction**（截断不会让前缀引用变不可达）。
- 附带：**M3-4 回归门自动触发**（输出 vs arXiv 基线对照 + ⚠️ + "非标准数据集/冒烟仅供参考"），端到端验证回归门 ✅。

## M3-3 结论
- **量级清楚**：thinking-off 下 arXiv brief ~¥2–4（阈内）；富多源 brief 原 ~¥7–13，**body 上限后 ~¥3.5–7**（接近/进 ¥5 阈）；P50 在 10min 内。
- **首要杠杆已落实**：analyze body 上限把富多源单跑 $1.86→$0.99；进一步收紧 / 内容感知阈值留观察。
- **干净定稿**（Anthropic 列表价、Sonnet、缓存命中）待直连 key；当前为 Opus-on-relay provisional 标定。
